# SolarFlare
Move the asteroids in front of the earth to save the day.
<h3>Overview</h3>
2D Game with mouse drag to drag the asteroids.
<br />
Features:
<br />* Particle collisions
<br />* Mouse drag movement
<br />* A basic health bar
